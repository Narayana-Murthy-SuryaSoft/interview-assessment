### Prerequisites

- Python3
- mutagen
  - Command
  ```sh
  pip install mutagen
  ```

### How to run

Run the commands as specified in the `README.md`. Replace `<program>` with `python3 music_manager.py`.

For example:

```sh
<program> --input <path to music folder> --list-files
```

becomes

```sh
python3 music_manager.py --input <path to music folder> --list-files
```
